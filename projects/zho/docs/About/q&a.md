#
以下是关于misspy的问题的解答。
## 问：版本控制方法是什么？
A. misspy 使用与 misskey 相同的日历版本控制。
## 问：我想同步运行一个函数
A. 目前应该是异步完成的。
## 问：我发现了一个漏洞。
答：[在此处]报告（https://github.com/sonyakun/misspy/security/advisories/new）。
## 问：支持哪些版本的 Misskey？
如下。

| 版本 | 支持 | MiAuth |
| :--------------------: | ------- | -----: |
| v11 及更早版本（包括 v11 及更早版本的分支）| ❌ | ❌ |
| v12.27.0 之前 | ⭕️ | ❌ |
| v12.27.0 或更高版本 | ⭕️ | ⭕️ |
| v13 或更高版本（推荐）| ⭕️ | ⭕️ |
